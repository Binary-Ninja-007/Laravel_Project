# Livewire Charts

Neat Livewire Charts for your Laravel projects.

## Installation

You can install the package via composer:

```bash
composer require asantibanez/livewire-charts
```

## Usage

```blade
<livewire:livewire-line-chart
    :line-chart-model="$lineChart"
/>
```

### Testing

``` bash
composer test
```
